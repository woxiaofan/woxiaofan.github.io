hexo+github博客搭建简易步骤

发表于 2017-01-06

搭建个人博客

  • npm install hexo-cli -g 安装工具
  • mkdir dirname 创建blog目录(不创建的话默认为创建日期命名文件名)
  • cd dirname 进入目录
  • hexo new ‘file name’ 新建md文件
  • hexo init 初始化hexo
  • npm install 安装依赖
  • npm install hexo-deployer-git –save 安装git发布插件
  • 创建github仓库，用户名.github.io
    • 配置 _config.yml，添加如下：
      • deploy:
        • type: git
        • repo: github地址（ssh地址）
        • branch: master
  • hexo g 生成页面
  • hexo s 启动本地预览服务器
  • hexo d 发布到github
  • hexo n ‘name’ 新建blog
  • 访问路径 用户名.github.io
